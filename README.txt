A Pen created at CodePen.io. You can find this one at https://codepen.io/alexG08/pen/rYbEWO.

 You can easily edit the TypeWriter class attributs like the first delay, the speed of typing (the lower, the faster), the delay after each line and the final delay after pressing the enter key. You can add lines in the linestoDisplay array from the main class.
This pen launch itself automatically after uncommenting the typeWriter.start() method, on line 86 in the JS file.
